How it works

  1. 1
    Install the companion app Download the desktop agent and pair it with your account — it records your stage runs and submits them. Get the app →
  2. 2
    Join a club Pick a club below. Each club's championships lay out events — a set of stages, open for a window, in the cars they allow.
  3. 3
    Set a time Ready to set a time? Start your run in the companion app, then run the event's stage in-game. The agent posts your time and you land on the leaderboard.

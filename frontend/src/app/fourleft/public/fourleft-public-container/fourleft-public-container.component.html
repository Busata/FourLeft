Fourleft

Fourleft is a Dirt Rally 2.0 project, offering a discord bot that:

  • Notifies channels when a new time is set for a club
  • Offers commands to show the results or standings for a club
  • Adds an extra layer to clubs via configuration, so clubs can be merged, filtered, divided into tiers
  • Add powerstages to results
  • Rank badges
  • Posts the daily/weekly/monthly community results, for those who opted in to do so

Contact @Busata#9432 on Discord for more information.
